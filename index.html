ChatApp

Author: Diya Patel

Last Update: 2 April, 2021 Sprint 8

Description:

This is a GUI based ChatApp that usees network programming to connect server and the client under same network. It will use Java Sockets to handle network connections. Using sockets,a chat server will be created between one server and a client.

Non-Prioritized List of Features:

  • GUI for server side
  • Adding More buttons on chat window
  • Make it work for one server, and many clients

    • Language : Java

    Flatform: Visual Studio Code

  • Features of Object Oriented Programming: Inheritance, Encapsulation, Abstraction, Polymorphism
  • Event Handling
