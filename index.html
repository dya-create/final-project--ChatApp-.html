Web Extractor

Author: Diya Patel

Last Update: 24 April, 2021 Sprint 7

Description:

A tool that gathers data from a website. It will parse data publish the data as a text. There will be a window where the user can enter a URL and press the button. When the button will be pressed, the application will read the URL and the the data will be displayed. The user will have a option to exit the program by clicking exit button.

Non-Prioritized List of Features:

  • Fetch data from multiple pages
  • Displaying data that is more visually appealing
  • Addiing more buttons with more functions

    • Language : Java

    Flatform: Visual Studio Code

    Technical Features that Demonstartes Things Learned in Class

  • Graphical User Interface
  • Features of Object Oriented Programming: Inheritance, Encapsulation, Abstraction, Polymorphism
  • Storing data in an ArrayList
